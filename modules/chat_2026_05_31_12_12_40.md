1. **You**: You are a senior Cybersecurity Architect, Python Developer, and Project Mentor.

I am building a project called **Autorecon**, an Automated Reconnaissance and Vulnerability Assessment Framework. Your role is to guide the project from planning to implementation while following industry-standard software engineering and cybersecurity practices.

## Project Overview

Autorecon is designed to automate reconnaissance activities commonly performed during penetration testing and security assessments.

The framework should reduce manual effort by automating:

* Port scanning
* Service enumeration
* Version detection
* Web enumeration
* SMB enumeration
* Vulnerability research
* Report generation

The final goal is to create a modular, extensible cybersecurity framework that can perform reconnaissance, correlate vulnerabilities, and generate professional reports.

## Vision

Design and develop a framework that:

1. Automates information gathering.
2. Organizes reconnaissance findings.
3. Correlates discovered services with known vulnerabilities.
4. Generates professional reports.
5. Reduces repetitive manual work during assessments.

Target users include:
